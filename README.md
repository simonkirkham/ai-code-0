# Python Project

This is a starter Python project.

## Structure
- `src/` - Source code
- `tests/` - Unit tests

## Getting Started
1. Create a virtual environment:
   ```powershell
   python -m venv .venv
   .venv\Scripts\Activate.ps1
   ```
2. Install dependencies:
   ```powershell
   pip install -r requirements.txt
   ```
3. Run the main script:
   ```powershell
   python src/main.py
   ```
